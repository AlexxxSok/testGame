## Technologies
- Svelte 3.x
- Typescript
- SCSS
- ESLint
- Prettier
- whatwg-fetch
- Webpack
- nodejs

## Development

1. Install dependencys

```console
npm install
```
2. Star development server
```console
npm run start
```
3. Start to develop
Navigate to [localhost:5000](http://localhost:5000).
